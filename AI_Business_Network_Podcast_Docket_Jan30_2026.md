## 📰 THE STORY

Anthropic launched two products, one this month, that are changing how work gets done—and they're flying under the radar compared to the flashier announcements from Google and OpenAI.

**Claude Code** is an agentic coding tool that lives in your terminal or for us non-coders lives in the desktop or online apps. It understands your entire codebase, executes commands, handles git workflows, writes and runs tests, and submits pull requests—all through natural language. It works in your terminal, VS Code, Cursor, JetBrains IDEs, and you can even tag @claude on GitHub issues. Dario Amodei said at Davos that some Anthropic engineers have stopped writing code entirely—they just direct Claude Code.

**Cowork** launched January 12th as "Claude Code for the rest of your work." It's built into the Claude Desktop app and lets you designate a folder where Claude can read, organize, and modify files autonomously. Think: reorganizing your cluttered downloads folder, generating expense spreadsheets from receipt screenshots, drafting reports from scattered notes, or analyzing data across multiple documents. It runs in a secure virtual machine on your Mac, and Anthropic built the entire feature in about 1.5 weeks—using Claude Code itself.

## 💼 WHY IT MATTERS (Business Impact)

**For Founders:**
- These tools represent the shift from "AI as chatbot" to "AI as autonomous worker." The paradigm is changing from asking AI questions to delegating AI tasks.


**For Executives:**
- This is what "agentic AI" looks like in practice—not theoretical, but shipping today. Evaluate how your teams could use these tools for immediate productivity gains.


**For GTM Operators:**
- **Sales:** Use Claude Code to quickly build custom demos, analyze competitor codebases, or automate reporting scripts. Non-technical sellers can direct Claude to build tools they need.
- **Marketing:** Cowork can process campaign data, organize creative assets, generate reports from multiple sources, and clean up messy content libraries.
- **RevOps:** Automate data cleaning, CRM exports, and analysis workflows. Point Cowork at your downloads folder full of CSV exports and let it make sense of the chaos.
- **Content:** Draft documents from research notes, organize source materials, batch-process files for publishing workflows.

## 🛠️ HOW TO USE THEM (Practical Guide)

### Claude Code Setup (For Technical Users & Curious Non-Coders)

**Installation:**
```bash
npm install -g @anthropic-ai/claude-code
```

**Basic usage:**
- Navigate to any codebase in your terminal
- Type `claude` to start
- Ask in natural language: "Explain how authentication works in this project" or "Write tests for the user service" or "Create a PR that fixes the bug in issue #47"

**Power moves:**
- Use in VS Code/Cursor: Install the Claude Code extension, highlight code, ask questions or request changes
- Tag @claude on GitHub issues to have it investigate, propose solutions, or submit PRs
- Run multiple Claude Code instances in parallel on different tasks (works great on mobile connected to a cloud VM)

**Best for:** Developers, technical founders, anyone who touches code even occasionally

### Cowork Setup (For Everyone)

**Access:**
- Download Claude Desktop app (Mac only currently, Windows coming soon)
- Subscribe to Claude Pro ($20/month) or Max ($100/month)
- Cowork is now available to Pro, Max, Team, and Enterprise plans

**Basic usage:**
1. Open Claude Desktop
2. Click "Cowork" mode
3. Select a folder you want Claude to work with
4. Describe what you want done in natural language

**Example prompts:**
- "Organize my Downloads folder by file type, rename files to be descriptive based on their content, and delete anything older than 6 months"
- "Look at all the receipt images in this folder and create an expense spreadsheet with date, vendor, amount, and category"
- "Read all the meeting notes in this folder and create a summary document with action items organized by person"
- "Analyze the CSV files in this folder and create a report on the trends you find"

**Power moves:**
- Point it at your messy project folders before client calls
- Use it to prep research—drop articles/PDFs into a folder, ask for a synthesis
- Batch-process content: "Convert all these markdown files to a single formatted Word doc"

**Best for:** Operations, marketing, sales, anyone who works with files and documents

## 📰 THE STORY

2025 was the year AI learned to browse the web. Every major AI company—and several startups—launched browser agents that can autonomously navigate websites, fill forms, compare products, and complete multi-step tasks. This week, Google entered the race with Auto Browse in Chrome, the first mainstream deployment to billions of users. But they're late to a party that's been building all year.

Here's the complete timeline:

### THE AGENTIC BROWSER TIMELINE

| Product | Company | Launch Date | Current Status |
|---------|---------|-------------|----------------|
| **Manus** | Butterfly Effect (Monica) | March 6, 2025 | Being acquired by Meta ($2-3B) |
| **Dia** | The Browser Company | June 11, 2025 (beta) / Oct 9, 2025 (public) | Mac only, acquired by Atlassian ($610M) |
| **Perplexity Comet** | Perplexity | July 9, 2025 (paid) / Oct 3, 2025 (free) | Mac, Windows, Android |

### WHAT EACH ONE DOES

**Manus** — The autonomous generalist. Not a browser per se, but an AI agent that can research topics, navigate websites, fill forms, generate content, deploy code, and run tasks asynchronously without human intervention. Built on Claude and fine-tuned Qwen models. Scored 86.5% on GAIA benchmark, outperforming OpenAI's DeepResearch. Now being acquired by Meta for $2-3B with Chinese regulatory review pending.

**Dia** — The "AI-native browser" from Arc's creators. URL bar doubles as AI chatbot. Can summarize files, search across all open tabs, and write drafts based on tab contents. Features "Skills"—user-created and built-in shortcuts for tasks like summarization, fact-checking, and productivity planning. Atlassian acquired The Browser Company for $610M; Arc features coming to Dia.

**Perplexity Comet** — The search-first browser. Perplexity's AI search engine is pre-installed and default. Comet Assistant lives in a sidecar, can see web pages and answer questions. Max subscribers ($200/month) get "background assistant" for multi-task delegation: booking flights, sending emails, finding deals. Now free globally, available on Mac, Windows, and Android.

**Claude for Chrome** — The extension approach. Adds Claude to a sidecar in Chrome that maintains context of everything in your browser. Can take actions with permission. Max plan only ($100-$200/month). Anthropic reduced prompt injection attack success from 23.6% to 11.2% through security research.

**ChatGPT Atlas** — OpenAI's full browser replacement. ChatGPT built into the browser core. "Agent Mode" can perform multi-step tasks: adding ingredients from recipes to Instacart, comparing e-commerce listings, executing workflows with consent. "Browser memories" remember context from visited sites (30-day retention). Free features for all, Agent Mode for Plus/Pro ($20+/month).

**Google Chrome Auto Browse** — The mass-market play. Coming to billions of Chrome users. Gemini 3-powered agent handles vacation planning, appointment scheduling, form filling, quote gathering, expense reports, subscription management. Co-developing Universal Commerce Protocol (UCP) with Shopify, Etsy, Wayfair, Target. AI Pro ($20/month) or Ultra ($250/month) required. U.S. only initially.

## 💼 WHY IT MATTERS (Business Impact)

**For Founders:**
- The browser is becoming an AI battleground. If you're building a web-based product, it now needs to work for both human AND AI visitors.
- The UCP standard Google is creating with major retailers could become mandatory. Get ahead of this now.
- Startup opportunity: security and governance layers for agentic browsing. Prompt injection attacks are a real threat that enterprises will pay to mitigate.

**For Executives:**
- Your employees will be using these tools within months, whether you sanction them or not. Create a policy now.
- Evaluate how AI agents experience your web properties—both your marketing site and your product. This is the new customer journey audit.
- Consider the data implications: agents like Atlas store "browser memories" on company servers. What does that mean for enterprise use?

**For GTM Operators:**
- **Sales:** These agents can research prospects, summarize their digital footprint, and fill out outbound forms at scale. Test which one works best for your workflow.
- **Marketing:** Your SEO strategy needs to account for AI intermediaries. When agents browse on behalf of users, traditional landing pages may be bypassed entirely.
- **Ops:** Procurement could be transformed. Agents comparing vendors, filling RFPs, gathering quotes—this changes the buying process.
